HTML

HTML (Hypertext Markup Language) é a linguagem de marcação padrão usada para criar páginas da web e aplicativos da web. Ela fornece uma estrutura para conteúdo na World Wide Web, usando um sistema de elementos e atributos para definir o layout e o conteúdo de um documento. Os elementos HTML são representados por tags, que os navegadores interpretam para renderizar os elementos visuais e auditivos de uma página da web. A linguagem evoluiu por meio de várias versões, com HTML5 sendo o padrão atual, introduzindo elementos semânticos, suporte multimídia aprimorado e controles de formulário aprimorados.
