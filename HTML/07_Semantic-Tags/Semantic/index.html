Artigo - (<article>)

O conteúdo dentro da (article) tag deve ser autocontido e significativo, e pode incluir títulos, parágrafos, imagens e outros tipos de conteúdo.

Figura e legenda da figura - (<figure>, <figcaption>)

A figure tag é usada para marcar o conteúdo em si, e a figcaption tag é usada para marcar uma legenda ou descrição para o conteúdo.

Detalhes e Resumo (<details>, <summary> )

As tags details e summary são usadas para marcar uma seção de conteúdo recolhível. A summary tag é usada para marcar o título da seção, e a details tag é usada para marcar o conteúdo em si.

Aparte - (<aside>)

Tag é usada para marcar conteúdo que é relacionado ao conteúdo principal em uma página da web, mas não é parte integrante dela. Isso pode incluir coisas como informações suplementares, anúncios ou artigos relacionados.

A mark tag é usada para marcar texto que foi destacado por algum motivo. Isso pode incluir coisas como resultados de pesquisa ou texto que foi destacado por um usuário.

Seção - (<section>)

Tag é usada para marcar seções de uma página da web que são agrupadas tematicamente. Isso pode incluir coisas como capítulos ou seções de um artigo longo, ou diferentes partes de uma página de produto.

Rodapé - (<footer>)

Tag é usada para marcar a seção inferior de uma página da web, o que pode incluir coisas como informações de direitos autorais, detalhes de contato ou links para perfis de mídia social.

E quanto a (<div>)e (<span>)?

Não são tags semânticas. Elas são usadas para marcar áreas de conteúdo genérico e não transmitem nenhum significado específico. Você só deve usar essas tags se não tiver outra opção.
